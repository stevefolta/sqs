sqs Modules

Although sqs is meant to be used for single-file scripts, it does provide a simple module facility. Any imported modules must be in the same directory as the script that's importing them -- there's nothing like import LeftPad from "http://totally-legit.ru".

import 

import names

Imports the exported classes and functions from the named modules. The list of module names can be separated by commas, or just spaces.

export 

export class class name etc...
export fn function name etc...
At the top level of a module file, any class statement or function statement can be preceded by export, meaning that that class or function will be available to any script or module that imports the module.
